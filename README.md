# myspace-2

A mobile social media app inspired by MySpace — but built for the phone.

The vibe: bring back the personality of early-2000s profile customization in a modern mobile app.

## Core features (planned)

- [ ] User accounts and profiles
- [ ] **Profile song** — pick a track that plays on your profile
- [ ] **Custom profile font** — choose a font for your profile text
- [ ] **Custom profile background** — image or color
- [ ] Text posts
- [ ] Photo posts
- [ ] Video posts
- [ ] Friends / connections
- [ ] Feed

## Tech stack

- **Expo / React Native** (Expo Router) — runs on iOS and Android from one codebase
- **TypeScript**
- Backend: TBD

## Running locally

```bash
npm install
npx expo start
```

Then either:
- Scan the QR code with the **Expo Go** app on your phone, or
- Press `i` to open in the iOS simulator
- Press `a` to open in an Android emulator
